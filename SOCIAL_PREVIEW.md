# Social Media Preview Setup

## Option 1: GitHub Repository Settings (Recommended)
1. Go to: https://github.com/Jon-Hodl/Bitcoin-Price/settings
2. Scroll down to "Social preview"
3. Click "Edit"
4. Upload an image (recommended: 1280×640px)
5. GitHub will automatically generate Open Graph tags

## Option 2: Custom Open Graph Image
The `og-image.svg` file in this repo can be converted to PNG:
- Recommended size: 1200×630 pixels
- Upload to repository root as `og-image.png`
- The HTML already references this URL

## Option 3: Use Online Tool
Create a custom OG image at:
- https://ogimage.net/
- https://www.opengraph.xyz/

## Current Setup
The HTML includes Open Graph meta tags that will display when shared on:
- Twitter/X
- Facebook
- LinkedIn
- Discord
- Telegram

Meta tags reference: `https://jon-hodl.github.io/Bitcoin-Price/og-image.png`
